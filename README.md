# Spring Boot User Management

A RESTful user management API built with Spring Boot

## Features

- Create a user
- Get a user by ID
- Get all users
- Search a user by email
- Update a user
- Delete a user
- Filter users by active status
- Request validation
- Duplicate email handling

## Tech Stack

- Java
- Spring Boot
- Spring Web
- Spring Data JPA
- Jakarta Validation
- Lombok
- MySQL
